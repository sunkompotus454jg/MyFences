# 🟦 MyFences
> *Стильная и легковесная утилита для Windows, созданная для идеального порядка на рабочем столе.*

**MyFences** позволяет создавать прозрачные, сворачивающиеся "сетки" (окна) для группировки ярлыков, файлов и папок в духе киберпанка и минимализма. 

Программа написана на **Python** с использованием библиотеки **PyQt6** и работает поверх рабочего стола на нижнем уровне Z-Order *(не перекрывает другие открытые окна)*.

---

## ✨ Главные фишки

* 🧠 **Умное сворачивание:** Окна автоматически сворачиваются в тонкую шапку, когда мышь уходит за их пределы, и плавно разворачиваются при наведении.
* 🧲 **Магнитное прилипание (Snapping):** Сетки аккуратно магнитятся друг к другу и к краям экрана (дистанция 20 пикселей) для создания идеального пиксельного порядка.
* 🎨 **Глубокая кастомизация:** 5 встроенных неоновых пресетов (Синий, Фиолетовый, Зеленый, Оранжевый, Красный) + возможность создать **свой собственный пресет** по HEX-коду через встроенную палитру с предпросмотром.
* 🛡️ **Безопасность файлов:** При удалении сетки все файлы из неё автоматически и безопасно перемещаются обратно на Рабочий стол.
* ⚙️ **Интеграция с Windows:** Программа добавляет пункт *"Создать Сетку (Fences)"* прямо в контекстное меню рабочего стола и автоматически запускается вместе с системой.
* 🖐️ **Drag & Drop:** Файлы легко перетаскиваются внутрь сетки обычным движением мыши.

## 🚀 Установка

Поскольку программа упакована в полноценный инсталлятор, установка занимает всего пару кликов:

1. Запустите файл `MyFences_Setup.exe`.
2. Следуйте инструкциям на экране. Установщик автоматически добавит программу в Автозагрузку и пропишет контекстное меню.
3. **Готово!** Программа уже работает в фоновом режиме.

## 📖 Инструкция по использованию

### 1. Как создать новую сетку
* Кликните **Правой кнопкой мыши (ПКМ)** по пустому месту на вашем Рабочем столе Windows.
* В появившемся меню выберите пункт **"🟦 Создать Сетку (Fences)"**.
* На экране появится новая базовая сетка.

### 2. Управление файлами
* **Добавить:** Просто перетащите любой файл, папку или ярлык с рабочего стола внутрь развернутой сетки.
* **Открыть:** Кликните **Дважды** левой кнопкой мыши по иконке внутри сетки.

### 3. Настройка окна
* **Перемещение:** Наведите мышь на название сетки, зажмите левую кнопку мыши и тащите. Окно будет "магнититься" к другим окнам.
* **Изменение размера:** Наведите мышь на правый нижний угол развернутой сетки (там появится неоновый уголок), зажмите левую кнопку и потяните.
* **Переименование:** Кликните **Дважды** левой кнопкой мыши по текущему названию окна, введите новый текст и нажмите `Enter`.

### 4. Дизайн и Цвета (Темы)
Кликните **Правой кнопкой мыши** по пустому месту *внутри* развернутой сетки, чтобы открыть меню кастомизации:
* **🎨 Цвет этого окна:** Меняет стиль только для текущей сетки.
* **🌍 Цвет всех окон:** Мгновенно применяет выбранный стиль ко всем сеткам на рабочем столе.
* **➕ Создать свой пресет...:** Открывает окно, где вы можете ввести название, выбрать цвет контура и фона. Этот пресет сохранится навсегда!
* *Удаление пресета:* Если вам разонравился ваш кастомный цвет, откройте меню тем и кликните по нему **Правой кнопкой мыши**, чтобы удалить.

### 5. Удаление сетки
* Откройте меню (**ПКМ** внутри сетки) и нажмите **"❌ Удалить сетку"**.
* Окно закроется, а все файлы, которые в нем лежали, будут аккуратно выгружены на ваш Рабочий стол.

## 🛠 Для разработчиков (Сборка из исходников)

Если вы хотите собрать проект самостоятельно.

**Требования:**
* Python 3.x
* PyQt6 (`pip install pyqt6`)
* PyInstaller (`pip install pyinstaller`)
* Inno Setup *(для компиляции установщика `setup.iss`)*

**Сборка `.exe`:**
Откройте консоль в папке проекта и выполните команду:
```cmd
pyinstaller --noconsole --onefile --name="MyFences" --icon=icoc.ico main.py

Затем скомпилируйте файл installer_script.iss через программу Inno Setup.

Создано с душой и любовью к чистому рабочему столу. 💙
